# markdown 이란? 
 - 텍스트를 서식 있는 문서로 쉽게 변환 할 수 있는 마크업 언어.
 - HTML으로 변환하기 쉽게 설계되었다.

# 마크업 언어란?
 - 텍스트에 태그를 사용하여 문서의 구조와 서식을 정의하는 언어. 
 - 문서를 체계적으로 작성하고 읽을 수 있도록 함.(HTML, XML....)

```
    백틱(`) <-- 3개 코드라인 markdown 문법이 적용안되는 영역
```
